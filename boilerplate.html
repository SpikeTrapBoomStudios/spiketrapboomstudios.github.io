Boilerplate

This page is all about boilerplate.

Boilerplate is any length of code that serves as a sort of template. It is common to use in web development if you want all of your pages to look very similar. Or, in game development when you have a player script that you reuse in every project, or an item script that for some reason you don't want to make a class for.

Boilerplate basics:

Boilerplate actually draws from the plates steel workers used in making... boilers. Since they need consistency and repetition, they made plates that would quickly do the forming process. boilerplate is fun.

This image is of a boiler. That's it, nothing else to it. However, its a nice visual for a page about boilerplate.

Boilerplate can be many things. A line of code... or a big file of code, and it is very helpful either way. You may be wondering why I am making a page about boilerplate. Well because this page is the boilerplate file for this website. Pull up any other page and you will see the similarities.
